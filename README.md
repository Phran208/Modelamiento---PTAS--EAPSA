# Modelamiento PTAS EAPSA - 

Repositorio con los códigos de simulación para el Informe Técnico de la planta EAPSA (Melipilla).

## Descripción
Este proyecto contiene scripts en Python para simular:
1. **Desempeño Biológico:** Cinética de Monod y proyección de DBO5 al 2026.
2. **Falla Hidráulica:** Simulación de *Washout* en decantadores secundarios ante eventos de lluvia.

## Archivos
* `Codigos_EAPSA_Temática_2.ipynb`: Notebook principal con todos los gráficos (1 y 2).

## Librerías Necesarias
El código requiere las siguientes librerías de Python:
* numpy
* matplotlib
* scipy
